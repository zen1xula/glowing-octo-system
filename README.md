# Sunrise GOS Helper — веб-версия (для телефона, с автообновлением)

Это обычный сайт: свежие тексты кодексов подтягиваются **сервером** сам по себе
(каждые несколько часов + при каждом старте сервера), а с телефона ты просто
открываешь ссылку в браузере — никакой установки не нужно.

## Как развернуть бесплатно (Render.com, ~10 минут)

1. Зарегистрируйся на https://render.com (можно через GitHub).
2. Залей эту папку к себе на GitHub (создай новый репозиторий и загрузи туда файлы —
   можно прямо через веб-интерфейс GitHub, кнопка "Add file → Upload files").
3. В Render: **New → Web Service** → выбери свой репозиторий.
4. Настройки:
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Plan**: Free
5. В разделе **Environment** добавь переменную (нужна только для ИИ-поиска):
   - `ANTHROPIC_API_KEY` = твой ключ с https://console.anthropic.com (API Keys)
   - Опционально: `REFRESH_HOURS` = как часто обновлять базу в часах (по умолчанию 6)
6. Нажми **Create Web Service**. Через пару минут появится ссылка вида
   `https://sunrise-gos-helper.onrender.com` — открой её с телефона и добавь на
   главный экран (в браузере: "Добавить на экран Домой"), будет как приложение.

Бесплатный тариф Render "засыпает" после ~15 минут без запросов и просыпается
за 30-50 секунд при следующем заходе — для личного пользования это некритично.
Если нужно, чтобы не засыпал совсем, — на Render есть недорогой платный план (Starter),
или можно использовать другой хостинг (Railway, Fly.io — процесс аналогичный).

## Локальная проверка перед деплоем

```
npm install
npm start
```

Открой http://localhost:3000 — сервер сразу начнёт первое скачивание базы.

## Обновление списка законов

`src/codexSources.js` — список тем форума (`name`, `code`, `url`). Добавляй новые
законы туда же; если на форуме вышла новая редакция и поменялся URL — обнови ссылку.

Если форум изменит вёрстку и парсер перестанет находить текст — правится в
`src/updater.js`, функция `fetchThreadFirstPostText` (селектор `.message-body .bbWrapper`).

## Что где хранится

- Тексты кодексов — на сервере, в `data/codexes.json`, обновляются автоматически
- Избранное и заметки — в браузере телефона (localStorage), у каждого свои
